# Kirjautumissivu ja admin-hallintapaneeli

1. Asenna Node.js koneellesi: https://nodejs.org/

2. Avaa komentorivi ja mene projektikansioon.

3. Asenna riippuvuudet:
   npm init -y
   npm install express

4. Käynnistä serveri:
   node server.js

5. Mene selaimella osoitteeseen:
   http://localhost:3000

- Admin salasana: admin123
- Käyttäjätiedot tallentuvat users.json tiedostoon.
